# salGO
Egy jelenleg privát saját fejlesztésű discord bot ami A és B(FIX) pont közötti autós utazások szervezését segíti.
## Backstory
A OE-NIK levelező képzése Salgótarjáni kihelyezésben zajlik és mivel a többség Budapest és vagy környéki így jó páran autót használunk a közlekedésre. Ezeknek a szervezése a szerver szobákban elkavarodhat így jött az ihlet a salGO bot elkészítésére.
### Elnevezés
Az elnevezés Salgótarján és az angol go (megy) szóbol tevődikössze játékosan, így egyszerre fejezi ki a célt és az akciót is.
